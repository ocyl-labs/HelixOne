Brand Usage Guidelines

✅ Do's
• Use the full logo with adequate spacing
• Maintain color gradient integrity
• Keep minimum size of 30px height
• Use on contrasting backgrounds
• Apply consistent brand colors
❌ Don'ts
• Don't distort or stretch the logo
• Don't change the helix colors
• Don't use on busy backgrounds
• Don't add effects or shadows
• Don't separate brand elements
📏 Spacing
• Minimum clear space: logo height ÷ 4
• Padding around icon: 15% of width
• Text spacing: 1.5x letter height
• Logo isolation zone required
• Consistent margins on all sides
📱 Digital Use
• SVG format for web (scalable)
• PNG for social media
• Minimum 400x400px for profiles
• Optimize for dark/light modes
• Test on various screen sizes
